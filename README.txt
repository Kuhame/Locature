Utilisation :

Pour pouvoir lancer notre projet, il faut suivre les étapes ci-dessous :
	
	1) Importer la base de données (bdd.sql) sur un serveur MySQL comme phpMyAdmin.
	2) Modifier le fichier connect.php dans le dossier m et mettre les identifiants pour vous connecter à votre serveur MySQL.
	3) Exécuter votre serveur web (UWAMP, LARAGON, ...) depuis la racine du projet ( dans le répertoire 'Version MVC (complete)' ).
	4) Ouvrir le fichier index.php sur le navigateur de votre choix.
